# Portfolio

My Web Portfolio

# Build With

- React
- Tailwind CSS
- styled-components

# Installation

1. Clone the repo
<pre><code>git clone https://github.com/Victorhbm/portfolio.git</code></pre>

2. Install NPM packages
<pre><code>npm install</code></pre>

3. Start the app
<pre><code>npm start</code></pre>
